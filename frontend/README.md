# QuSim Frontend

## Setup

1. Install dependencies:
```bash
cd frontend
npm install
```

2. Start the development server:
```bash
npm start
```

The frontend will run on http://localhost:3000

## Features

- **Visual Circuit Editor**: Drag-and-drop gates to build circuits
- **Code Editor**: Write circuits using DSL syntax
- **Synchronization**: Visual and code editors stay in sync
- **Interactive Gates**: Click gates to select, drag to move, delete button when selected
- **Wire Visualization**: See connections between gates

## Usage

1. Click a gate in the palette to add it to the circuit
2. Drag gates to reposition them
3. Click a gate to select it (shows delete button)
4. Switch between Visual and Code tabs to see both representations
5. Click "Execute Circuit" to run the circuit
